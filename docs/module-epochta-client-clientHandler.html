clientHandler

epochta-client~ clientHandler

Singleton. Instance could be instantiated without new operator. Implements all API methods provided by ePochta SMS service API.

Important:

  • All methods arguments are positional! To provide optional argument, omitting previous optional arguments use null as a stub.
     //example
 client.getAddressbook(null, 10, 10).then...
  • All methods returns Promise
  • All Promises could be rejected with APIError or ResponseCodeError or any error returned by Request module
