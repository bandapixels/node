Inrerface for methods provided by ePochta SMS service API.
All arguments are positional! To provide optional argument, omitting previous optional arguments use null as stub.

Example

Use null as stub 

// Ways to get first ten address books:
correct: client.getAddressbook(null, null, 10).then...
wrong: client.getAddressbook(10).then...
